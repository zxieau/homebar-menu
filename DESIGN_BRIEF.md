# DESIGN_BRIEF.md

## Product
Jimmy's Bar 是一个给朋友来家里点单用的 home bar 手机网页，后续可能会考虑使用 iPad 来操作（先不考虑相关界面优化）。

## Visual Direction
- 黄昏后的英伦街道感
- 私人酒吧、低调但有质感
- 像一张被风吹到手里的酒单传单
- 有轻微纸张质感、手绘符号、边角磨损，但内容清晰
- 页面主体必须是 menu 本身，背景只服务氛围

## Do
- mobile-first
- 使用清晰的酒单层级：酒名 / 简短描述 / 风味标签 / 下单按钮
- 多用细节：纸张纹理、微弱阴影、手写小标注、分隔线
- 留白要自然，不要空洞
- 元素之间要有节奏感

## Don't
- 不要做成企业官网
- 不要做成 SaaS dashboard
- 不要大面积渐变玻璃卡片
- 不要使用廉价霓虹灯风格
- 不要普通圆角卡片一列排到底
- 不要 emoji 代替设计
- 不要过度装饰影响可读性

## Color
- 主色：暖黄、昏橙、深棕、墨黑
- 辅色：暗红或橄榄绿可以少量使用
- 禁止：高饱和蓝紫渐变、科技感青色

## Typography
- 标题：有一点复古感、像酒单标题
- 正文：必须清晰易读
- 英文小字可以有 printed menu / handwritten note 的感觉

## Layout
- 首页优先展示菜单
- 点单流程尽量轻，不要复杂后台感
- 所有交互都要适合朋友现场用手机点单
